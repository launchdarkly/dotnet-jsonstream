Remarks

This method returns a new, modified ObjectReader. It should be called before the first time you call Next(JReader). For instance:

Copy
var requiredProps = new string[] { "key", "name" };
for (var obj = reader.Object().WithRequiredProperties(requiredProps); obj.Next(ref reader);)
{
    switch (obj.Name) { ... }
}

When the end of the object is reached, if one of the required properties has not yet been seen, Next(JReader) will throw a RequiredPropertyException.

For efficiency, it is best to preallocate the list of property names globally rather than creating it inline.
