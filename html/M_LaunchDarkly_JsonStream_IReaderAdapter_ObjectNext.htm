Remarks
This method will only be called if NextValue(NullableValueType, Boolean) previously returned Object. It should update the adapter's state so that the next call to NextValue(NullableValueType, Boolean) will return the value of the next property. If there are no more properties, it should return None and update the adapter's state so that the current position is after the end of the object.
