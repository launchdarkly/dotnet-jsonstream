Remarks

The specified type must implement IJsonStreamConverter. This allows the JsonStreamConvert methods to work with the target type:

Examples
Copy
[JsonStreamConverter(typeof(MyConverter))]
public class MyType { ... }

public class MyConverter : IJsonStreamConverter
{
    // implement IJsonStreamConverter methods to handle MyType
}

var instanceToEncode = new MyType();
var someJson = JsonStreamConvert.SerializeObject(instanceToEncode);

var decodedInstance = JsonStreamConvert.DeserializeObject<MyType>(someJson);

On platforms where System.Text.Json is available, this attribute also works as an equivalent to the [JsonConverter] attribute, so System.Text.Json.JsonSerializer methods will also work with the target type and will delegate to the specified converter.
