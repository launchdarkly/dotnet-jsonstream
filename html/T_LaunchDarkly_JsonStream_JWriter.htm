Remarks

On platforms that provide the System.Text.Json API (.NET Core 3.1+, .NET 5.0+), this works as a wrapper for System.Text.Json.Utf8JsonWriter, providing a more convenient API for common JSON writing operations. On platforms that do not have System.Text.Json, it falls back to a portable implementation that is not as fast as Utf8JsonWriter but still highly efficient and has no external dependencies. Portable JSON marshaling logic can therefore be written against this API without needing to know the target platform.

The general usage pattern is as follows:

  • There is one method for each JSON data type.
  • For writing array or object structures, the Array and Object methods return a struct that keeps track of additional writer state while that structure is being written.
  • If any method encounters an error, an exception is thrown and the caller should not attempt to use the JWriter any further after that point.
Examples
// This writes the following JSON object: {"a":true,"b":[1,2]} var w = JWriter.New(); var obj = w.Object(); obj.Name("a").Bool(true); var arr = obj.Name("b").Array(); arr.Int(1); arr.Int(2); arr.End(); obj.End(); var json = w.GetString();
