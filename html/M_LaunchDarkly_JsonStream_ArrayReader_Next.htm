Remarks

It returns if the JReader has reached the end of the array, or if the array was empty or null.

If Next returns , you can then use JReader methods such as Bool to read the element value. If you do not care about the value, simply calling Next again without calling a JReader method will discard the value.

For more information about why Next requires a ref parameter, see JReader.
