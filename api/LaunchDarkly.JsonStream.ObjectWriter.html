Class ObjectWriter

A decorator that manages the state of a JSON object that is in the process of being written.

public sealed class ObjectWriter : ValueType, IDisposable
Remarks

Calling Object() creates an ObjectWriter. Use Name(String) or MaybeName(String, Boolean) to begin writing each object property, and IValueWriter methods to write each value. Calling End() or Dispose() terminates the object.

Type Description
IValueWriter

an IValueWriter which may or may not be enabled
Remarks

If present is true, this behaves the same as Name(String). However, if present is false, it does not write a property name and instead of returning a functional IValueWriter, it returns a stub object that does not produce any output.

Examples
    var obj = writer.Object();

    // this writes a property "present":"value"
    obj.MaybeName("present", true).String("value");

    // this writes nothing
    obj.MaybeName("absent", false).String("value");
