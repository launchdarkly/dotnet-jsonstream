Class ArrayReader

Used in conjunction with JReader to iterate through a JSON array.

Inheritance
System.Object
ArrayReader
Namespace: LaunchDarkly.JsonStream
Assembly: LaunchDarkly.JsonStream.dll
Syntax
public sealed class ArrayReader : ValueType
Remarks

The Array() and ArrayOrNull() methods return an instance of this type. Call the Next(ref JReader) method to iterate through each array element. To read the actual values, you will still use the JReader's methods.

Examples
    var r = new JReader("[10, 20, 30]");
    for (var arr = r.Array(); arr.Next(ref r);)
    {
        value = r.Int();
        System.Console.WriteLine(value);
    }

Remarks

It returns false if the JReader has reached the end of the array, or if the array was empty or null.

If Next returns true, you can then use JReader methods such as Bool() to read the element value. If you do not care about the value, simply calling Next again without calling a JReader method will discard the value.

For more information about why Next requires a ref parameter, see JReader.
